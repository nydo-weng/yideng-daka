# 1.请你总结如何计算算法时间复杂度+空间复杂度。

**时间复杂度**:  
算法执行所需要的操作数

- 没有循环: 把每行代码本身需要的操作数相加. 保留最高次数项, 去系数.
- 有循环: 进入循环的次数 \* 循环内的操作数, 如果有嵌套循环, 继续相乘. 同样保留最高次数项, 去系数.
- 有条件判断/操作数随输入内容变化而变化, 通常取最坏的情况.

**空间复杂度**:  
算法执行所需要的内存空间

- 算法执行所需的内存数量固定, 比如常数个变量, O(1)
- 需要空间和输入成正比, 比如要复制一份输入, 则为 O(n)
- 多维数组, O(n^n)
- 递归, 乘上递归的深度

# 2.什么是栈？JavaScript 中有哪些栈的具体应用？

- 栈是后进先出的数据结构, 常见操作包括, 入栈, 出栈, 查看栈顶, 判断是否为空. 在 JS 中可以用数组来模拟栈的逻辑结构.
- 在 JS 运行的时候, 内存中也会分配一块区域为调用栈(call stack). 用于执行同步代码, 函数的执行上下文等会被推入栈中, 等函数结束返回时, 就出栈.
